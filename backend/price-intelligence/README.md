# OnlyPropFirms - Price Intelligence Module

## Overview
This module handles all logic related to:
- Prop Firm Source Catalog (Where to get data)
- Price Tracking & History
- Deal Detection (Changes/Discounts)

## Quick Start
1. **Frontend:** Types are in `frontend/src/types/pricing.ts`
2. **API:** Spec in `openapi/pricing.openapi.yaml`
3. **Docs:** See `docs/OVERVIEW.md`
